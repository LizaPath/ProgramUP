# Учебная практика УП.02

## Выполнила: Патрушева
## Проект: Информационная система управления спортивной секции

### Описание проекта
Данный репозиторий содержит исходный код информационной системы, предназначенной для автоматизации учёта студентов, записанных в спортивные секции университета. 
Система позволяет вести реестр спортсменов, управлять их контактными данными (Email) и распределять их по спортивным группам (GroupName).

### Этапы разработки
В рамках данного репозитория выполняется полный жизненный цикл разработки ПО:
1. Проектирование архитектуры и базы данных.
2. Реализация модулей работы с данными и бизнес-логики.
3. Тестирование, отладка и оценка качества по стандартам ISO.
4. Подготовка технической и сертификационной документации.
   
### Используемые технологии
* Язык программирования: C#
* Фреймворк: ASP.NET Core MVC
* ORM: Entity Framework Core
* База данных: Microsoft SQL Server
* Система контроля версий: Git / GitHub
